# AI 投資小秘書

## 功能
- AI 根據年齡與風險偏好建議 ETF 定期定額組合
- 20 年複利模擬曲線
- **模擬 LINE 簡訊**（畫面即時顯示，無需帳號）

## 技術亮點
- `yfinance` 抓真實歷史報酬
- 欄位自動偵測（`Adj Close` / `Close`）
- 錯誤處理與預設值，提升穩定性
- **模擬簡訊**：方便評審體驗

## 外部資源說明
- **yfinance**：Yahoo Finance 歷史資料
- **LINE Messaging API**（模擬）：官方文件 [developers.line.biz](https://developers.line.biz/)
